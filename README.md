# MEGALISP'S WEBSITE
> Pre-V1

### Where we're at:
Version 1 is mostly just working with existing [Newt](https://github.com/megalisp/newt), not-so cleverly breaking things up into a bunch of inject-whatever.html files that we load in our existing template. V2 (likely mid-to-late 2026) we want to move to using [parasite](https://github.com/megalisp/parasite) as much as possible and be the driver of its development. Basically everything should be in an s-exp and/or parasite form, and we export to whatever instead of needing to direcvtly mess with 3+ different syntaxes as we do now (html, css, json, javascript, etc)
